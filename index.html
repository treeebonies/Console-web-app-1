Visual Studio Code is a source-code editor made by Microsoft for Windows, Linux and macOS. Visual Studio Code is a streamlined code editor with support for development operations like debugging, task running, and version control.

Console shows where there can be any errors in your written code, and what be adjusted. In order to openm the console screen, right click on the desired tab and choose the inspect option. Then, at the top bar, choose the console option. To print something on cnsole, in your code write console.log("") and within parenthesis and quoteations (or without quotes depending on what your writing) and the quotes text will be inputed to console
